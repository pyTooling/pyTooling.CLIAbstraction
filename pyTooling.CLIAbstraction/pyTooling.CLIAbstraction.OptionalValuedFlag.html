class pyTooling.CLIAbstraction.OptionalValuedFlag.OptionalValuedFlag(value=None)[source]Â¶

Class and base-class for all OptionalValuedFlag classes, which represents a flag argument with data.

An optional valued flag is a flag name followed by a value. The default delimiter sign is equal (=). Name and value are passed as one arguments to the executable even if the delimiter sign is a whitespace character. If the value is None, no delimiter sign and value is passed.

Example: width=100
